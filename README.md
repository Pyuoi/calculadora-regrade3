Rule of Three Interactive Calculator
Overview

    This is a web-based tool designed to solve direct and inverse proportion problems. The application features a responsive interface and real-time validation to ensure mathematical accuracy and a smooth user experience.

Features

    Calculation Modes: Supports both Direct and Inverse proportion logic through a toggle interface.
    
    Input Validation: Includes checks for empty fields and prevents division by zero.
    
    Dynamic Styling: Provides visual feedback via CSS state changes when errors are detected.
    
    Responsive Design: Optimized for different screen sizes and devices.

Technical Specifications

    Frontend: HTML5 and CSS3 using Flexbox for layout positioning.
    
    Logic: Vanilla JavaScript (ES6+) implementing modular arrow functions.
    
    DOM API: Comprehensive use of Document Object Model for event handling and element manipulation.
